          # 原生JS使用

          使用npm或yarn安装

          然后引入到项目中即可

          import ZPlayer from 'zooplayer'

const player = new ZPlayer({
    el: "", // 指定挂载的选择器
    video: {
        src: "" // 视频播放地址
    }
})

player.mount()

          # Vue中使用

          使用npm或yarn安装

          全局注册

          import ZPlayer from "./player"

Vue.use(ZPlayer);

          可参考提供原生JS和Vue版的Exampleopen in new window
